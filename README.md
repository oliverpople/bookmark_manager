##USER STORIES


```
As a user
So that I can quickly go to web sites I regularly visit
I would like to see a list of links on the homepage

As a user
So I don't have to remember a URL
I want to be able to save a new bookmark

As a user
So I can categorise my links
I want to be able to add tag(s) to my bookmarks

As a user
So I can search through my bookmarks
I want to be able to filter the list by tags
```
